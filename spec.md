## Implementation Plan
1. Minimal Motoko backend (no real backend features needed — purely frontend experience)
2. Single-page React app with scroll-based section navigation
3. CSS animations: starfield particles, glitch effects, pulsing glows, fog layers
4. Canvas or CSS 3D sphere for hero
5. Tailwind + custom CSS variables for deep black/indigo/gold/cyan palette
6. Google Fonts: Cinzel or Playfair Display (serif) + Rajdhani or Orbitron (sans-serif UI)
7. Framer Motion or pure CSS for parallax and scroll animations
8. Reality Shift toggle using React state to swap lore text between Grid/True Reality versions
9. Hover cards for lore reveals
10. Section-by-section build with consistent visual language
